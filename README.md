## Problem
Campus event ticketing systems are centralized and prone to fake tickets, duplicate entries, and unauthorized reselling.

## Solution
A decentralized ticketing system built on Algorand where each ticket is issued as a non-transferable NFT. Ownership of the NFT proves entry.

## Key Features
- NFT-based tickets
- Non-transferable (soulbound) tickets
- On-chain ticket verification

## Tech Stack
- Algorand Blockchain
- PyTeal
- HTML, CSS, JavaScript
